# Stat-o Design System & Styleguide

## Brand Concept
Stat-o is built on a foundation of structural logic, offering precise and minimal data visualization. 
The brand seeks to remove noise by utilizing clean geometry, unified spacing, and a clear, focused color palette.

## Core Palette
The Stat-o theme has been simplified from a multi-color spectrum into a mono-accent orange palette, emphasizing visual calm and clarity.

- **Primary Accent / Mild Orange**: `#FF7F50` (Used for active states, borders, and main graph highlights).
- **Secondary / Dark Orange**: `#E06031` (Used for typography on light orange backgrounds and secondary chart areas).
- **Surface / Background**: Off-white `#FDFDFD` with a diffused mesh gradient of `#FF7F50` layered on top.
- **Text & Core Outlines**: Deep Charcoal (`#374151`) and Dark `#1F2937` to guarantee contrast readability.

## Typography
Stat-o uses an interwoven stack focusing on geometric, highly legible forms.

- **Primary Stack**: `Satoshi, Inter, sans-serif`
- **Headings**: Tracked tight, Extrabold (`800`) weight.
- **Body & Captions**: Medium/Bold for UI emphasis, tracking slightly wider for technical readouts.

## UI Component Rules
- **Grid Layout**: Constructed on an 8px base rhythm.
- **Border Radius**: Cards and panels adhere to a unified base radius of `12px` (`--radius-xl` / `-2xl` / `-3xl`).
- **Shadows**: Large area, soft transparency drops. (`0 4px 20px rgba(0,0,0,0.03)`).
- **Gradients**: Data visualizations map to solid fills fading into `.1` opacity white space, directing visual flow top-down.
